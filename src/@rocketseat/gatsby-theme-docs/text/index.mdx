# What is it Squzy

[![version](https://img.shields.io/github/v/release/squzy/squzy.svg)](https://github.com/squzy/squzy)

Squzy it is a high-performance open-source monitoring, incident and alert system.

## Features

- 💨 Self host solution - we not have access for your data
- 📝 HTTP/HTTPS site monitoring (v1.0.0+)
- 🛣 TCP host monitoring (v1.0.0+)
- 🥇 Json value monitoring (get value by json path) (v1.3.0+)
- 🖥 SSL Expiration - validate expiration of SSL certificate (v1.11.0+)
- 📱 gRPC application monitoring via [protocol](https://github.com/grpc/grpc/blob/master/doc/health-checking.md) (v1.0.0+)
- 🖥 SiteMap monitoring (each url inside sitemap) (v1.0.0+)
- 🥇 Json value monitoring (get value by json path) (v1.3.0+)
- 📈 Get host information CPU/Memory/Disks/Net via agents (v1.7.0+)
- 📄 Application monitoring: request/tracing/databases/frontend (v1.8.0+)
- ⚡️  Incident with incident lang (v1.9.0+)
- 🔔 Notification manager (v1.10.0+)
- and much more 🔥

## Why you can use us?

- We not collect any data on our servicies, all products fully opensource without any hidden part. You can install in your cloud/servers/systems and all data will be with you.
- We have important core functional for monitoring system and that will be improved more soon.
- We have uniq incident lang, where you can define any rules. [Examples](https://demo.squzy.app/agents/5f142b6141a1c6518723483a/live)
- With increasing the number of users the quality of product will be grow too, we really think opensource best case for provide monitoring system
- We have open protocol, so, if you want to use your own implementation for example for storage, just implement it.
- We fully free without any payment.
