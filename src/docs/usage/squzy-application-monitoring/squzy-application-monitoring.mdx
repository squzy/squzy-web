# About

Squzy Application Monitoring server which collects information from application.
Right now it takes transaction information + tracing (between servicies).

[Demo: tracing + gin + grpc](https://demo.squzy.app/transactions/Y_jLC4hlwirv0PYqvSVG5)

[Github DEMO](https://github.com/squzy/test_tracing)

[![version](https://img.shields.io/github/v/release/squzy/squzy.svg)](https://github.com/squzy/squzy)
