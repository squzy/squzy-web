[Demo](https://demo.squzy.app/notification-methods/list)

## API

We use Protobuff for describe API

[GRPC API](https://github.com/squzy/squzy_proto/blob/master/proto/v1/squzy_notification_server.proto)

## Support types

### Webhook

Method: *POST*

```go
type WebhookRequest struct {
	Id        string             `json:"id"`
	Status    api.IncidentStatus `json:"status"`
	CreatedAt string             `json:"createdAt,omitempty"`
	UpdatedAt string             `json:"updatedAt,omitempty"`
	Link      string             `json:"link,omitempty"`
}
```
