## About

Squzy Agent Server is a part of Squzy which deal with agent information handling.
It receives information from all the system agents, after what sends it to Squzy Storage for saving.

[![version](https://img.shields.io/github/v/release/squzy/squzy.svg)](https://github.com/squzy/squzy)
