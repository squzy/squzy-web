## About

Squzy Storage is a part of squzy, which stores the most part of inforamtion: application, agent, monitoring data and incidents.

By this moment we are using postgress.

[![version](https://img.shields.io/github/v/release/squzy/squzy.svg)](https://github.com/squzy/squzy)
